| Command                              | Purpose                                                 |
| ------------------------------------ | ------------------------------------------------------- |
| `npx prisma init`                    | Initialize Prisma                                       |
| `npx prisma migrate dev --name init` | Create and apply a migration                            |
| `npx prisma generate`                | Generate the Prisma Client                              |
| `npx prisma db push`                 | Push schema to the database without creating migrations |
| `npx prisma studio`                  | Open the Prisma Studio GUI                              |
| `npx prisma migrate reset`           | Reset the database and reapply migrations               |
| `npx prisma migrate deploy`          | Apply existing migrations in production                 |
